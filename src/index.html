Timestamp microservice

-Pass unix timestamp it will convert to natural date and vice versa

-It will display both unix timestamp and natural date

-It will return null for unknown parameters

Example

Input

https://love-timestamp-microservice.herokuapp.com/January 1, 2016

https://love-timestamp-microservice.herokuapp.com/1451606400


Output

{"unix":1451606400,"natural":"January 1, 2016"}
